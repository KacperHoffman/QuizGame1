## Quiz Game


# O Aplikacji

Aplikacja jest grą opartą na zasadach quizu. Gracz ma do wyboru ponad 20 różnych kateogorii oraz 3 poziomy trudności gry. 
Jedna gra składa się z 10 pytań. Gra jest w pełni responsywna.
